El presente documento recoge el análisis forense, evaluación de vulnerabilidades y proceso de hardening realizado sobre un servidor Debian comprometido en la empresa 4Geeks Academy. El objetivo del proyecto fue identificar el origen del incidente de seguridad, detectar vulnerabilidades presentes en el sistema, aplicar medidas correctivas y validar la seguridad final del entorno.
Durante el desarrollo se aplicaron técnicas de análisis de logs, enumeración de servicios, explotación controlada de vulnerabilidades y configuración de mecanismos de protección, siguiendo buenas prácticas de ciberseguridad y marcos de referencia como NIST SP 800-61 e ISO 27001.
